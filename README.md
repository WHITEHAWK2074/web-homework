# 元宵节·潮汕普宁网页作品
项目基于 HTML + CSS 制作，用于展示广东揭阳普宁地区元宵节的传统习俗、文化元素和现场场景。

## 项目简介
- 作品类型：静态响应式网页
- 开发内容：HTML 结构搭建、CSS 样式布局、图片视频展示
- 展示内容：元宵节习俗、传统活动、风景图片、民俗场景等
- 用途：课程作业、个人作品展示、文化宣传

## 技术说明
- HTML5 语义化标签
- 图片与视频资源已压缩优化，保证加载速度
- 使用 GitHub Pages 在线部署

## 页面包含内容
- 首页介绍：元宵节主题与普宁地区文化背景
- 习俗介绍：营老爷，英歌舞，放烟花等传统活动
- 风景与实景图片：潮汕春节氛围、夜景、花灯、街头场景
- 民俗视频：活动记录、花灯巡游（已压缩优化加载）
- 文化说明：简要介绍普宁元宵节文化特色

## 优化说明
- 图片已通过压缩工具减小体积，提升加载速度
- 视频采用 720P 分辨率，确保清晰度同时控制文件大小
- 所有路径改为相对路径，保证在线可访问
